# Getting Help

If you encounter any issues or need some help while working with the API, you can reach out to Lightspeed’s API support team in one of the following ways:

## For 3rd party developers

Please see our [help center](https://retail-support.lightspeedhq.com/) for all product questions. If you’re still unable to find an answer, use the ‘contact us’ button in the help center to request further support.

## For Lightspeed Partners

As Lightspeed partners, you have access to the API support team’s inbox. Your partner manager should have provided you with the email address, if not, reach out to them for more information. When you email API support, please try to include as much of the following information as possible:

- Name of integration:
- Name of technical contact:
- Email address of technical contact:
- Name of affected account:
- AccountID of affected account:
- Description of the issue:
- API Request URI:
- Payload being sent:
- Error message from Lightspeed (if applicable):
