# TagGroup

> Example XML Read

```xml
count="0">
  
    example1
  
  
    example2
```

#### Description

This is a helper control to get a list of tag groups for the current account.

#### Actions Allowed

| GET/Read (single) | False |
| --- | --- |
| GET/Read (query) | True |
| POST/Create | False |
| PUT/Update | False |
| DELETE/Delete | False |

#### Scopes Required

| employee:cfd | true |
| --- | --- |
| employee:register | true |
| employee:inventory_base | true |
| employee:reports | true |
| employee:workbench | true |
| employee:customers | true |
| employee:admin | true |

#### Sortable Fields

This endpoint is not sortable.
